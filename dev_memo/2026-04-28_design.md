# 設計メモ — WBS ↔ Outlook 同期ツール

作成日: 2026-04-28

---

## システム概要

```
WBS_Sample.xlsx  ←→  wbs_outlook_sync.py  ←→  ローカルOutlook (COM)
```

Outlook連携は **pywin32 (win32com.client)** を使用。Azure AD / Graph API 不要。
Outlookがインストールされているマシン上で動作。

---

## WBS 列定義（add-time-cols 適用後）

| 列 | 名称 | 説明 |
|---|---|---|
| A (1) | ProjectName | プロジェクト名 |
| B (2) | ProjectID | プロジェクトID |
| C (3) | Phese | フェーズ |
| D (4) | L1_Category | 大分類1 |
| E (5) | L2_Major_Classification | 大分類2 |
| F (6) | L3_Middle_Classification | 中分類 |
| G (7) | L4_Task | タスク名 |
| H (8) | TaskID | タスクID（数値） |
| I (9) | Assinged Person | 担当者名 |
| J (10) | AP_MailAddress | **ID（Microsoft 365メールアドレス）** |
| K (11) | StartDate_p | 開始予定日 → Outlookへ送る |
| L (12) | StartTime_p | 開始予定時刻 (HH:MM) ★追加★ |
| M (13) | FinishDate_p | 終了予定日 → Outlookへ送る |
| N (14) | FinishTime_p | 終了予定時刻 (HH:MM) ★追加★ |
| O (15) | StartDate | 開始実績 ← Outlookから受け取る |
| P (16) | FinishDate | 終了実績 ← Outlookから受け取る |
| Q (17)〜 | ガントチャート | 日付ヘッダー + IF式 |

## Outlookイベント設計

- **件名フォーマット**: `[WBS][{TaskID}] {L4_Task}`
  - 例: `[WBS][1] システム要件記入`
- **本文**: TaskID / プロジェクト名 / タスク名 / 担当者 など
- **種類**: 通常の予定（AllDayEvent = False）
- **マッチングキー**: 件名の正規表現 `\[WBS\]\[(\d+)\]` で TaskID を抽出

## コマンド一覧

```bash
# 初回のみ: 既存WBSに時間列を追加
python wbs_outlook_sync.py add-time-cols --file WBS_Sample.xlsx

# WBS → Outlook（イベント作成/更新）
python wbs_outlook_sync.py wbs2outlook --file WBS_Sample.xlsx --id user@example.com

# Outlook → WBS（実績日程を書き戻し）
python wbs_outlook_sync.py outlook2wbs --file WBS_Sample.xlsx --id user@example.com [--output 出力先.xlsx]

# 新規WBS作成（ガントチャート範囲指定）
python wbs_outlook_sync.py new-wbs --output new_WBS.xlsx --gantt-start 2026-04-01 --gantt-end 2026-06-30
```

## 同期フロー

### wbs2outlook
1. WBSを読み込み、AP_MailAddress == 指定IDの行を抽出
2. ローカルOutlookのカレンダーを開き、`[WBS]`タグ付きイベントをTaskIDでキャッシュ
3. 各タスクについて:
   - StartDate_p + StartTime_p → イベント開始日時
   - FinishDate_p + FinishTime_p → イベント終了日時
   - TaskIDが既存イベントにあれば **更新**、なければ **新規作成**

### outlook2wbs
1. WBSを読み込み、AP_MailAddress == 指定IDの行を抽出 (TaskIDセット構築)
2. ローカルOutlookからWBSタグ付きイベントを取得
3. TaskIDが一致する行に Start/End を書き戻し
   - → StartDate 列 (O)、FinishDate 列 (P)

## 注意事項

- ファイルをExcelで開いたまま保存しようとするとエラーになる
- Outlookが起動していない場合、COM呼び出し時に自動起動する
- ガントチャートの数式: `=IF(AND({col}$1>=$K{row},{col}$1<=$M{row}),1,"")`
  - $K = StartDate_p, $M = FinishDate_p（add-time-cols適用後）
